# Context Directory Contract

Эта папка хранит контекст в виде файлов, пригодных для Mirage, Codex и обычных shell-команд.

## Правила

1. Сырые данные сначала попадают в `00-inbox`.
2. Устойчивые факты о пользователе и проектах фиксируются в `10-profile`.
3. Описания подключенных каналов и их схемы хранятся в `20-sources`.
4. Машиночитаемые извлечения пишутся в `30-extracted`.
5. Короткие агентные сводки и индексы пишутся в `40-summaries`.
6. Устаревшее переносится в `90-archive`, а не удаляется без причины.

## Минимальный стандарт файла

Каждый важный файл должен отвечать на четыре вопроса:

- источник;
- дата или диапазон;
- статус проверки;
- что агенту разрешено делать с этим материалом.
